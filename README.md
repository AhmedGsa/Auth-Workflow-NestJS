## Description

This repo contains a JWT Based authentication with NestJS and MySQL Database.

Don't forget to add a .env file with the following variables (Your MySQL DB Configuration):

- JWT_SECRET (An encryption key)
- DB_NAME
- DB_USERNAME
- DB_PASSWORD
- DB_PORT

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
